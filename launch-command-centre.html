Launch order

Do these steps in this order

  • Upload the contents of the GitHub folder to the kronatrix-authors repository.
  • Make sure CNAME contains bookmarketing.kronatrix.co.uk.
  • Check GitHub Pages is serving the custom domain over HTTPS.
  • Open /robots.txt and /sitemap.xml in the browser.
  • Submit the sitemap in Google Search Console and Bing Webmaster Tools.
  • Request indexing for the homepage, /how-to-market-your-book.html, /free-author-ai-visibility-check.html and /author-website-examples.html.
  • Test all Formspree forms with your own email.
  • Add footer backlinks from all live proof websites.
  • Publish three external articles that mention KRONATRIX Authors naturally.
  • Review Search Console impressions weekly and improve pages that get impressions but no clicks.
