# GitHub setup for bookmarketing-kronatrix

Repository: `bookmarketing-kronatrix`

Live custom domain: `bookmarketing.kronatrix.co.uk`

## Upload instructions

Upload every file in this flat folder to the root of the GitHub repository. Do not upload the ZIP itself.

The repository root should contain files such as:

- `index.html`
- `how-to-market-your-book.html`
- `free-author-ai-visibility-check.html`
- `CNAME`
- `robots.txt`
- `sitemap.xml`
- `kronatrix-authors.css`
- `kronatrix-authors.js`
- the `.webp` image assets
- the `.json` data files

## GitHub Pages settings

Go to: Settings → Pages

Use:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

Custom domain:

`bookmarketing.kronatrix.co.uk`

## DNS record needed

In your DNS settings for `kronatrix.co.uk`, add:

Type: `CNAME`
Name: `bookmarketing`
Value: `kronatrixai.github.io`

## After launch

Submit this sitemap to Google Search Console and Bing Webmaster Tools:

`https://bookmarketing.kronatrix.co.uk/sitemap.xml`
